# Streaming Services User Analysis

## Project Overview
This project analyzes user behavior, engagement patterns, and revenue performance for a streaming platform.

The goal was to identify trends that can help improve:
- User engagement
- Subscription revenue
- Content recommendations
- Customer retention

## Tools Used
- - Excel (Dashboard & analysis)

## Dataset Features
The dataset contains user-level streaming platform information including:

- Subscription price
- Watch hours
- Movies & series watched
- Recommended content engagement
- Device usage
- Loyalty points
- Country & language preference

## Key Analysis Performed

### Subscription & Revenue Analysis
Analyzed revenue contribution by subscription tier.

### Engagement Analysis
Compared movies vs series consumption and average watch hours.

### Recommendation Impact
Studied correlation between recommended content and user engagement.

### Device Usage Trends
Identified most common streaming devices.

### Peak Watch Time
Analyzed when users consume the most content.

### Retention & Loyalty
Examined loyalty points distribution and membership status.

## Key Insights

- Premium users generate the highest revenue per user.
- Series consumption is higher than movie consumption.
- Evening and late-night hours show the highest platform engagement.
- Mobile and Smart TV dominate streaming devices.
- Users interacting with recommended content show significantly higher watch hours.
